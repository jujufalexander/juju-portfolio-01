Notable Projects

01. Ella Virtual Blockchain Assistant

The "Ella: Virtual Assistant for Blockchain Developers" project was developed using Python. The project incorporates Solidity as the language for the chatbot's smart contract generation tool, enabling developers to launch and customize prebuilt smart contracts directly from Visual Studio Code.


The project utilizes the Langchain library to ingest and index data, enabling the chatbot to interact with pre-defined content from loaded text documents and directories. This integration enhances the chatbot's ability to provide tailored and context-aware responses to users in the blockchain development domain.


Additionally, functions have been crafted to deliver customized cryptocurrency reports based on user preferences. These functions utilize external APIs to retrieve real-time data on cryptocurrencies. Users can specify their projects of choice, and the chatbot generates personalized reports with information such as current prices, percentage changes, and market capitalization.
